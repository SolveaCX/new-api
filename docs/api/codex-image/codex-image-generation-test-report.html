5. 观察与备注

计费自洽：三条记录的总费用均与「输入 token × $5.0505/M + 输出 token × $32.3232/M」精确吻合（A $0.006442 / B $0.05688 / C $0.031046），且计费来源为「上游返回」——证明按真实 image-gen token 计费，未走兜底。

edits 输入图计费：图生图的输入 token（291）显著高于纯文生图（~21），多出的 ~256 即输入图本身的 image-token——输入图成本被正确计入，符合预期。

定价提示：一张 high 质量图（输出 ~1756 token，$0.0569）约为默认质量（~196 token，$0.0064）的 9 倍。请据此设定 gpt-image-2 倍率，避免客户端自由请求高质量导致成本失控。

可改进项（非阻塞）：① response_format=url 拒绝当前返回 HTTP 500，语义上宜为 400；② 部分订阅档位上游可能将 high 实际降级为中等画质（账号档位行为，网关如实透传并按真实 token 计费）；③ n>1 仅返回 1 张（上游图像工具不接受 n，已在文档约定）。
